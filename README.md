# sql-data-warehouse--project
Built an end-to-end data warehouse by ingesting CSV data into the Bronze layer, applying ETL transformations in the Silver layer, and delivering curated, analytics-ready data models in the Gold layer.

=======================================
# 👋 WELCOME

Welcome to this Data Warehouse Project! This repository showcases the end-to-end implementation of a modern data warehouse.

=======================================

# 📌 About the Project

This project demonstrates building a scalable data warehouse using a layered architecture (Bronze, Silver, Gold). It covers data ingestion, transformation, and modeling to support analytical reporting and decision-making.

---

# ⚙️ Project Requirements

* SQL Server
* Basic understanding of SQL and data modeling
* CSV dataset (source data)
* Git (for version control)

---

# 🏗️ Building the Data Warehouse

The project follows a multi-layered approach:

* **Bronze Layer**: Raw data ingestion from CSV files using bulk insert
* **Silver Layer**: Data cleaning, transformation, and standardization (ETL process)
* **Gold Layer**: Final analytical model with fact and dimension tables

---

# 🎯 Objective

To design and implement a structured data warehouse that transforms raw data into meaningful insights for business analysis.

---

# 📐 Specifications

* Star Schema design (Fact & Dimension tables)
* One-to-Many relationships between entities
* Optimized SQL queries for performance
* Data validation and transformation logic
